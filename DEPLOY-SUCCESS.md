## 🎯 快速部署（推荐3步）

### 方案1：GitHub + Cloudflare Pages（最佳选择）

```bash
# 1. 推送到GitHub
git remote add origin https://github.com/你的用户名/statistics-learning-center.git
git push -u origin main

# 2. 在Cloudflare中连接仓库
# 访问：https://dash.cloudflare.com/pages
# 选择：Connect to Git → 选择你的仓库

# 3. 配置构建设置
# Framework preset: None
# Build command: (留空)
# Build output directory: .
```

### 方案2：直接上传（最快）

1. 访问：https://dash.cloudflare.com/pages
2. 选择："Upload assets"
3. 拖拽整个 `spss` 文件夹
4. 等待部署完成

### 方案3：Netlify（备选）

1. 访问：https://app.netlify.com/drop
2. 拖拽 `spss` 文件夹
3. 自动获得网站地址
