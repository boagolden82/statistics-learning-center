# 统计学习中心

一个帮助初学者轻松学习统计学核心算法的在线教育平台。

## 📚 项目介绍

统计学习中心专门为初学者设计，提供三种核心统计方法的深入学习：

- **线性回归** - 掌握数值预测和关系分析
- **逻辑回归** - 理解二分类问题和概率预测  
- **X平方检验** - 学习变量关联性检验

## 🎯 项目特色

### 🎨 现代化设计
- 响应式布局，支持各种设备
- 清晰的视觉层次和用户友好的界面
- 渐变色彩和现代化设计元素

### 📖 结构化学习
- **基础概念** - 直观的定义和解释
- **数学原理** - 详细的公式和推导过程
- **应用场景** - 实际案例和典型应用
- **交互演示** - 动态可视化和实时计算
- **练习题** - 巩固知识和自我测试

### 🔧 交互功能
- 线性回归可视化演示
- 逻辑回归概率计算
- X平方检验实时计算
- 进度跟踪和本地存储
- 答题反馈和结果验证

## 🚀 技术栈

- **前端框架**: HTML5 + CSS3 + JavaScript (ES6+)
- **UI框架**: Bootstrap 5.3
- **图标库**: Bootstrap Icons
- **数学公式**: CSS样式化公式显示
- **数据可视化**: Canvas API
- **本地存储**: LocalStorage API

## 🎨 设计理念

### 学习体验优化
- **渐进式学习**: 从基础概念到深入应用的递进式结构
- **可视化理解**: 通过图表和演示帮助理解抽象概念
- **即时反馈**: 练习题提供实时答案验证和解释
- **进度跟踪**: 记录学习进度，激励持续学习

### 内容组织
- **模块化设计**: 每个算法独立成页，便于专注学习
- **对比分析**: 专门的对比页面帮助理解差异
- **实践导向**: 丰富的应用案例和实际场景

## 📖 学习路径

### 推荐学习顺序
1. **首页概览** - 了解整体框架和目标
2. **线性回归** - 学习最基础的预测方法
3. **逻辑回归** - 理解分类问题解决方法
4. **X平方检验** - 掌握统计检验技术
5. **算法对比** - 理解三种方法的区别和选择
6. **实践练习** - 巩固所学知识

### 学习时间建议
- 每个算法学习时间: 30-45分钟
- 对比分析: 20-30分钟
- 练习题: 20-25分钟
- 总学习时间: 2-3小时
