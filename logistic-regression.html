什么是逻辑回归？

定义：逻辑回归是一种用于二分类问题的统计学习方法，它预测的是事件发生的概率，而不是具体的数值。
目标

预测二元结果，输出事件发生的概率

  • 垃圾邮件识别
  • 疾病诊断
  • 客户流失预测
特点

输出概率值，可解释性强，应用广泛

  • 概率解释直观
  • 算法简单高效
  • 理论基础扎实
核心思想

逻辑回归通过Sigmoid函数将线性回归的输出映射到(0,1)区间， 从而得到事件发生的概率。当概率大于阈值(通常是0.5)时，判定为正类；否则为负类。

数学原理

Sigmoid函数

逻辑回归的核心是Sigmoid函数，它将任何实数映射到(0,1)区间：

σ(z) = 1 / (1 + e⁻ᶻ)
当 z → ∞
σ(z) → 1
当 z = 0
σ(z) = 0.5
当 z → -∞
σ(z) → 0
逻辑回归模型
P(y=1|x) = σ(β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ)

其中：

  • P(y=1|x) - 给定特征x，y=1的概率
  • σ() - Sigmoid函数
  • β₀, β₁, ..., βₙ - 模型参数
  • x₁, x₂, ..., xₙ - 输入特征
决策边界

当P(y=1|x) = 0.5时，我们得到决策边界：

β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ = 0

决策边界将特征空间划分为两个区域，一侧预测为正类，另一侧预测为负类。

损失函数

逻辑回归使用对数损失函数(Log Loss)：

L = -Σ[yᵢ log(pᵢ) + (1-yᵢ) log(1-pᵢ)]

通过梯度下降等优化算法最小化损失函数来学习模型参数。

与线性回归的区别

对比维度 线性回归 逻辑回归
问题类型 回归 分类
输出范围 (-∞, +∞) (0, 1)
激活函数 无（恒等函数） Sigmoid函数
损失函数 均方误差(MSE) 对数损失(Log Loss)
应用场景 房价预测、销量预测 垃圾邮件检测、疾病诊断
关键区别：线性回归预测连续值，而逻辑回归预测概率。尽管名字叫"回归"，逻辑回归实际上用于分类问题。

应用场景

垃圾邮件检测

根据邮件内容、发件人等特征判断是否为垃圾邮件

特征：关键词频率、发件人信誉
目标：是否垃圾邮件(是/否)
医疗诊断

基于症状和检查结果预测疾病发生的概率

特征：年龄、症状、化验结果
目标：是否患病(是/否)
信用评分

根据财务历史预测客户违约风险

特征：收入、债务、信用历史
目标：是否违约(是/否)
客户流失

预测客户是否会停止使用服务或产品

特征：使用频率、满意度、消费金额
目标：是否流失(是/否)

本节要点总结
  • 逻辑回归用于二分类问题，输出事件发生的概率
  • 通过Sigmoid函数将线性组合映射到(0,1)区间
  • 使用对数损失函数进行参数优化
  • 决策边界将特征空间划分为两个类别
  • 广泛应用于医疗诊断、垃圾邮件检测等领域
